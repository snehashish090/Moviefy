About this Project:

This is a Web app Built using a Python Web Framework called Flask. Flask is an easy to work with framework in Python and thats why it is used in this web app as the app isn't that complex. The Project took about a week to develop and was a vey interesting project to work on. The app is a movie collection website where you can keep track of the movies you watch or want to watch. This can be useful if you watch a lot of movies and want to keep track of how many you watch in a month or if you want to see if you have watched a moive before if you can't remember.
